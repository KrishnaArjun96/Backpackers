Backpackers User Guide

This is the official user guide for the features available on Backpackers

Login

  • A user can use his username and password to log in to the application at this link
  • An account has been created for each user by the customer representative.
  • Depending on whether the user is a customer, manager or customer representative they are redirected to the relevant page that allows the user to perform actions like book flights and view existing reservations.

Customer help

  • The customer can perform many actions on the Backpackers interface.
  • The customer gets redirected to the index page upon login.
  • This index page can be used to search for flights.
  • The options available for searching flights is extensive.
  • The user can choose to fly from one destination to another with a direct flight or connecting flights.
  • The dates for the duration of the travel can be selected from this screen.
  • The user can also choose to travel with multiple passengers and in Economy,Business or First Class. Both these selections can be made with the help of a drop down list.
  • After making sure that all the fields are filled to the satisfaction of the customer, the customer can select Search to search for the relevant fights.
  • The results page has a list of flights that the customer can select. The user can choose flight with 0 or more stops. If the user wanted a return journey, the user would be able to select two flight plans.
  • Upon selecting the flight plans and selecting Go, the user is taken to the page with a summar y of the itinerary. The user can see the final price on this screen and their flight information
  • The user can also participate in a reverse auction and place a bid for each of the flights.
  • If the user chooses reverse auction by selecting the checkbox, a bid is placed with the amount input by the user.
  • The customer representative records a reservation if the bid is deemed acceptable.

Customer representative

  • A customer representative created by the manager can add,edit or delete user accounts.
  • The customer rep can also record a reservation if the customer has participated in a reverse auction.
  • The Customer rep is redirected to the relevant page when he logs in.
  • From this page the customer rep has multiple tabs that allow for certain actions to be completed.

Manager

  • A manager is responsible for creating, editing and deleting customer representative accounts. Upon logging in, the manager is takes to the relevant page where he can perform manager level transactions.
  • In the employee tab, the manager can see information about all the employees. The manager can choose to add,edit or remove employees.
  • In the Sales tab, the manager can see sales report information for different months and the employees responsible for each reservation.
  • In the flights tab, the manager can search for all flights and filter based on the the airline ID.
  • In the reservations tab, the manager can see the list of all reservations and the customers that booked these reservations. The manager can also see the flights were these reservations have been booked. In the revenue tab, the manager can see the revenue from each flight, revenue from each city, maximum revenue from a customer and maximum revenue by a customer representative.
